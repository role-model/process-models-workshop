7.1 Key questions

  1. How do I set up and run a RoLE model in R?

7.2 Lesson objectives

After this lesson, learners should be able to…

  1. Describe the structure of a RoLE model.
  2. Describe the rules of a RoLE simulation.
  3. Describe the outputs of a RoLE model and what they say about the system.
  4. Use roleR to run a basic RoLE model.
  5. Extract summary statistics from a RolE model, plot them, and relate the visuals to ecological/evolutionary processes.
  6. Save RoLE models to disk.

7.3.1 Installing roleR

You do not need to install roleR for this workshop, rather this is here for completeness and future information should you wish to install RoLE on your home cluster.

roleR is currently installed directly from the github repository using the remotes::github_install() function.

> install.packages("remotes")
> library(remotes)
> remotes::install_github('role-model/roleR')
